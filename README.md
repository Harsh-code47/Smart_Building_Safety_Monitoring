🏢 Smart Building Safety Monitoring

An IoT-based Smart Building Safety Monitoring System that continuously monitors environmental conditions using ESP32 sensor nodes, Raspberry Pi, MQTT, and a real-time monitoring dashboard.

🚀 Tech Stack

- ESP32
- Raspberry Pi 4
- MQTT (Mosquitto)
- Python
- SQLite
- HTML/CSS/JavaScript
